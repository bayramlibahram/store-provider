Overview

Store Provider is a powerful abstraction layer that provides a unified interface for managing data storage in web applications. It supports multiple storage backends including in-memory storage, localStorage, and sessionStorage.

Features

  • ✓ Multiple storage backends (memory, localStorage, sessionStorage)
  • ✓ Automatic fallback to memory storage when web storage is unavailable
  • ✓ Custom key prefixing for namespace isolation
  • ✓ Automatic JSON serialization/deserialization
  • ✓ Server-side rendering (SSR) compatible
  • ✓ TypeScript type definitions included

Installation

Copy store-provider.js to your project and import it as an ES module:

import StoreProvider from './store-provider.js';

Getting Started

Basic Usage

// Create a new store provider instance const store = new StoreProvider({ storeType: 'localStorage', prefix: 'myapp' }); // Store a value (objects are automatically serialized) store.setValue('user', { name: 'John', age: 30 }); // Retrieve a value (automatically deserialized) const user = store.getValue('user'); // Remove a value store.removeValue('user');

API Reference

Constructor

new StoreProvider(options) constructor

Creates a new instance of StoreProvider.

options.storeType
Type: "memory" | "localStorage" | "sessionStorage"
Default: "localStorage"
The storage backend to use.
options.prefix
Type: string
Default: "test-app"
Prefix for all storage keys to avoid collisions.
Example:
const store = new StoreProvider({ storeType: 'sessionStorage', prefix: 'myapp' });

Methods

setValue(key, value) method

Stores a value in the storage backend.

key
Type: string
The key under which to store the value.
value
Type: any
The value to store (will be JSON serialized).
Example:
store.setValue('settings', { theme: 'dark', notifications: true });

getValue(key) method

Retrieves a value from the storage backend.

key
Type: string
The key of the value to retrieve.
Returns
Type: T | string | null
The stored value (deserialized) or null if not found.
Example:
const settings = store.getValue('settings'); console.log(settings); // { theme: 'dark', notifications: true }

removeValue(key) method

Removes a value from the storage backend.

key
Type: string
The key of the value to remove.
Example:
store.removeValue('settings');

Examples

Using Memory Storage

const memoryStore = new StoreProvider({ storeType: 'memory', prefix: 'temp' }); memoryStore.setValue('cache', { data: [1, 2, 3] });

Using localStorage

const localStore = new StoreProvider({ storeType: 'localStorage', prefix: 'myapp' }); localStore.setValue('userPreferences', { language: 'en', timezone: 'UTC' });

Using sessionStorage

const sessionStore = new StoreProvider({ storeType: 'sessionStorage', prefix: 'session' }); sessionStore.setValue('tempData', { sessionId: '12345' });
